# garment-productivity-prediction
Machine Learning project for predicting garment employee productivity using Python. Includes data preprocessing, model training, and productivity prediction using XGBoost.
# Garment Productivity Prediction

This project predicts garment employee productivity using Machine Learning.

## Features
- Data preprocessing
- Missing value handling
- Label encoding
- Train-test split
- XGBoost model training
- Model evaluation
- Productivity prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost

## Dataset
Garments Worker Productivity Dataset
